# FinEdge India YouTube Channel Strategy

This project presents a 90-day YouTube channel growth strategy for FinEdge India, a personal finance and investing education platform.

- Build a strong YouTube presence
- Increase subscriber growth
- Improve audience engagement
- Implement effective video SEO strategies
- Create a sustainable content plan
